## [0.0.4] -- 2022-06-10

### Added
* lexemes
* tabs in morpheme and morph detail views

### Removed
* bugs

### Changed
* missing POS are rendered as `*`
* wordform structure links to morphemes even in detail view
* allow missing FormSlices in wordforms

### Added

* button for copying sentence IDs

### Removed

### Changed
